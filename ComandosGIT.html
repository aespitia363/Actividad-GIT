PRINCIPALES COMANDOS DE GIT

Comandos básicos

Existen muchos comandos que nos ayudarán a gestionar las diferentes tareas que queramos realizar con GIT, es importante denotar que los comandos siempre iniciaran con la palabra git, entonces tenemos git versión, que nos sirve para conocer la versión instalada de git, también está git help y git i, los cuales brindan ayuda con los comandos e inicia un nuevo repositorio respectivamente. Sin embargo, los comandos que más destacan son status, add . y commit.

  • git status:Este commando verifica y señala los archivos que han sido modificados.
  • git add .:Este sirve para registrar los cambios y agregarlos, de este modo quedará listo para hacer un commit.
  • git commit:Este comando es el que nos permite crear diferentes copias a las cuales se puede colver en caso de que sea necesario

Por otro lado, existen otros comandos que también son importantes y normalmente son usados en ámbitos en los que dos o más personas trabajan con el mismo documento, esto con el fin de poder reutilizar las diferentes versiones en caso de requerirlo. Por ejemplo, está el git log, que muestra una lista de los commit hechos, asimismo, es posible viajar a cada uno de ellos con el comando git reset.
